## Implemented Features

- 회원가입
- 로그인
- 미디어 목록 제공
    - 장르 별 제공
    - 추천 영상 제공
- 미디어 재생
    - 초대 링크 생성 및 복사
    - 미디어 재생 시간 동기화
    - 미디어 재생 속도 조절
    - 미디어 해상도 조절
    - 실시간 채팅
    - 이전 시청 기록에서 영상 재개
- 동영상 스트리밍 로그 저장
- 시청 기록 저장
- 동영상 스트리밍 로그 및 시청 기록 시각화
- 백엔드 REST API 문서 작성
- 설계 문서 작성
- 테스트 코드 / 문서 작성
